
Body:

```text
SeqWalk helps humans review AI-written code by turning execution paths into easy-to-read interactive HTML sequence diagrams.

This first release includes a native skill bundle, a strict HTML sequence viewer template, manual install guides, and adapters for Codex, Claude Code, Cursor, Gemini CLI, and other agent tools.
```

## Short Post

```text
AI can write code quickly, but humans still need to understand and review the execution flow.

SeqWalk is an open-source agent skill that turns code paths into interactive HTML sequence diagrams. It helps you see what calls what, where data moves, and which parts deserve review.

Works with Codex, Claude Code, Cursor, Gemini CLI, and similar AI coding tools.
```
